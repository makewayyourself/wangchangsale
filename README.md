선착순 왕창세일 (V16.0 Market Edition) 사용자 매뉴얼
이 매뉴얼은 게임 참여자(고객)와 관리자 모두가 시스템을 효과적으로 이해하고 사용할 수 있도록 제작되었습니다.
1. 게임 개요
"0.001초의 승부! 7.777초를 잡아라!"
이 프로그램은 스톱워치를 정확한 목표 시간(7.777초)에 멈추는 타이밍 챌린지 게임입니다. 성공 시 대박 포인트를 획득할 수 있으며, 실패하더라도 패자부활전(Lucky 7)과 위로금 쿠폰 등 다양한 혜택이 준비되어 있습니다.
2. 입장 및 로그인 (Login)
🅰️ 무료 입장 (VIP 구매자)
 * 대상: 제품을 구매하여 관리자로부터 아이디를 승인받은 고객.
 * 방법: 아이디 입력칸에 본인의 ID를 입력하고 GAME START 버튼 클릭.
 * 기회: 3회 (1라운드) 제공.
 * 특징: 게임 종료 시(성공 또는 3회 실패) 아이디는 자동 소멸됩니다. (재도전 시 관리자 재승인 필요)
🅱️ 유료 입장 (티켓 구매자)
 * 대상: 상점에서 '도전권'을 충전한 고객.
 * 방법: 티켓 충전 시 입력한 아이디를 입력하고 GAME START 버튼 클릭.
 * 기회: 구매한 티켓 매수만큼 제공 (50회 또는 100회).
 * 특징: 게임 도중 상단 배너의 [그만하기 (로그아웃)] 버튼을 눌러 남은 횟수를 저장하고 나중에 이어할 수 있습니다.
3. 게임 플레이 방법
 * 시작: START 버튼을 누르면 타이머가 빠르게 돌아갑니다.
 * 정지: STOP 버튼을 누르면 타이머가 멈춥니다.
 * 결과 확인:
   * 7.777초 (잭팟): 10만 R포인트 당첨! (즉시 종료 및 쿠폰함 지급)
   * 7.77x초 (성공): 1만 R포인트 당첨! (즉시 종료 및 쿠폰함 지급)
   * 꽝: 기회(하트/티켓)가 1회 차감되며 재도전 가능.
 * 특수 이벤트:
   * Lucky 7: 멈춘 숫자에 '7'이 3개 이상 포함되면(예: 1.777, 7.077) 보너스가 발동됩니다.
     * 티켓 유저: 보유 횟수에 즉시 보너스 횟수 추가.
     * VIP 유저: 50/100회 도전권 쿠폰 당첨 또는 생명력(+3회) 충전.
   * Near Miss: 목표 시간에 아주 근접하게 실패하면 "으악! 아까워요!" 팝업 효과가 뜹니다.
4. 상점 및 충전 시스템 (Shop)
 * 위치: 로그인 화면 하단 🎫 티켓 충전소 버튼.
 * 상품:
   * 50회 도전권 (3,000원)
   * 100회 도전권 (5,000원)
 * 구매 방식:
   * 충전할 아이디를 입력합니다.
   * 상품을 클릭하면 입금 계좌 정보**(국민 843101-04-394545 (주)씨드림코어)**가 팝업됩니다.
   * 입금이 확인되면 해당 아이디에 횟수가 충전됩니다. (현재 테스트 버전은 즉시 충전됨)
5. 선물 / 판매 기능 (Market) ✨
내 남은 횟수를 다른 사람에게 자유롭게 선물하거나 판매할 수 있는 기능입니다.
 * 이용 대상: 티켓을 구매하여 횟수를 보유한 유저 (무료 VIP 유저는 불가).
 * 사용 방법:
   * 게임 화면 중간의 🎁 선물/판매 버튼 클릭.
   * 받을 사람 아이디와 보낼 횟수 입력.
   * [보내기] 버튼을 누르면 내 잔여 횟수에서 즉시 차감되어 상대방에게 전송됩니다.
6. 시크릿 쿠폰함 (Secret Wallet)
이 게임의 핵심 보상 시스템입니다. 보안을 위해 쿠폰 코드는 처음에 가려져 있습니다.
 * 획득 경로: 게임 성공 (1등/2등), Lucky 7 당첨, 모든 기회 소진 시 (위로금)
 * 확인 방법:
   * 로그인/게임 화면의 📦 내 쿠폰함 버튼 클릭.
   * 목록에서 🔒 확인하기 상태의 쿠폰 클릭.
   * 주의: 클릭 시 "오픈하면 사용한 것으로 간주됩니다" 경고창이 뜹니다.
   * [확인]을 누르면 코드가 공개되고, 목록에서 즉시 삭제됩니다. (반드시 캡처하거나 즉시 사용하세요)
7. 관리자 기능 (Admin Only)
일반 사용자는 접근할 수 없는 운영자 전용 메뉴입니다.
 * 접속 방법: 로그인 화면 우측 상단 모서리의 빈 공간(히든 버튼) 클릭.
 * 비밀번호: 초기 설정은 1234입니다.
 * 주요 기능:
   * 승인 대기 명단 관리: VIP 무료 체험 아이디를 등록/삭제합니다.
   * 명단 업데이트(재충전): 게임이 종료된 VIP 유저의 아이디를 다시 입력하여 재도전 기회를 부여합니다.
   * 참여 로그(Log) 확인: 누가 언제 어떤 결과(잭팟, 쿠폰, 꽝 등)를 얻었는지 실시간 기록을 확인합니다.
⚠️ 주의 사항
 * 데이터 저장: 이 게임은 브라우저의 Local Storage를 사용합니다. 브라우저 캐시를 삭제하면 티켓 정보나 쿠폰함이 사라질 수 있으니 주의해 주세요.
 * 반응형: 모바일(세로), 태블릿, PC 환경 어디서든 최적화된 화면으로 즐길 수 있습니다.
